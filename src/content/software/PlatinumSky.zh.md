# PlatinumSky

**PlatinumSky** 讓您可以在 Mac OS 9 上 [BlueSky](https://bsky.app/)，享受社交焦慮，盡情催化你的多巴胺。

![PlatinumSky 發文](./PlatinumSky-screenshot.png "PlatinumSky 螢幕截圖")

## Requirements

- Mac OS 9
- PowerPC 處理器
- 32 MB RAM（建議 64 MB 以上）
- 可連線的網路（Open Transport TCP/IP）

## Features

- 瀏覽 timeline
- 可顯示圖片附件
- 回覆、轉發、喜歡、引用
- 發文可附圖片、可用表情符號
