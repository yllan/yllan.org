# OS9Map

**OS9Map** 讓您可以在 Mac OS 9 上看 [OpenStreetMap](https://www.openstreetmap.org/) 地圖。您也可以搜尋地標、地址，把感興趣的地點儲存成書籤，方便下次前往。

![OS9Map 顯示市中心街道圖](./OS9Map-screenshot.png "OS9Map 主視窗")

## Requirements

- Mac OS 9
- PowerPC 處理器
- 16 MB RAM（建議 32 MB 以上）
- 可連線的網路（Open Transport TCP/IP）

## Features

- **平滑捲動的地圖畫布** — 用滑鼠拖曳即可平移，邊捲邊載入鄰近的地圖磚。
- **地名搜尋** — 內建 Nominatim 查詢，輸入地址即可跳到對應位置。
- **書籤** — 把常去的地點存成書籤，從選單一鍵前往。
